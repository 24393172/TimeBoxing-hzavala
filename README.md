# ⏳ TimeBoxing - hzavala

Aplicación web fullstack para la gestión eficiente de tareas mediante la técnica de **Time Boxing**, permitiendo organizar actividades por bloques de tiempo y mejorar la productividad.

---

## 🚀 Tecnologías utilizadas

### 🎨 Frontend
- React
- Vite
- CSS
- JavaScript

### 🛠 Backend
- NestJS
- Node.js
- TypeScript

---

## 📁 Estructura del proyecto

TimeBoxing-hzavala  
│  
├── TimeboxFront   → Aplicación frontend  
└── back-nest      → API backend
